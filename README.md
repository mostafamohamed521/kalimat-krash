# كلمات كراش — Kalimat Crash
## هيكل المشروع

```
kalimat-crash/
│
├── index.html              ← شاشة البداية (Splash)
├── map.html                ← خريطة المستويات (Ocean Map)
├── game.html               ← شاشة اللعب (جميع المودات)
│
├── css/
│   ├── main.css            ← ستايل عام + splash screen
│   ├── map.css             ← ستايل الخريطة والمحيط
│   └── game.css            ← ستايل اللعبة (كل المودات)
│
├── js/
│   ├── main.js             ← JS الشاشة الرئيسية
│   ├── map.js              ← رسم الخريطة + الأمواج + الجزر
│   ├── game-engine.js      ← محرك اللعبة الكامل (4 مودات)
│   ├── levels-data.js      ← بيانات 100 مستوى
│   └── algorithms.js       ← خوارزميات CSP, Genetic, A*, etc.
│
└── assets/
    ├── images/
    │   ├── ui/
    │   │   ├── splash-bg.jpg     ← صورة خلفية شاشة البداية
    │   │   ├── ocean-bg.jpg      ← صورة خلفية الخريطة
    │   │   └── coin.png          ← أيقونة الكوينز
    │   └── levels/
    │       ├── l1.jpg            ← صور المستويات (l1 → l20)
    │       ├── l2.jpg
    │       └── ... l20.jpg
    └── sounds/                   ← (اختياري) أصوات اللعبة
```

## الصور المطلوبة

### `assets/images/ui/`
- `splash-bg.jpg` — خلفية شاشة البداية
- `ocean-bg.jpg` — خلفية خريطة المحيط
- `coin.png` — أيقونة العملة (اختياري)

### `assets/images/levels/`
- `l1.jpg` إلى `l20.jpg` — صور المستويات للمود الأول
- كل صورة تظهر مع كلمات معبرة عنها

## الخوارزميات المستخدمة
- **CSP + Backtracking** — لتوليد جريد الكلمات المتجاورة (Mode 1, 2)
- **Constraint Satisfaction** — لوضع الكلمات في Word Search (Mode 3)
- **A* Search** — لإيجاد مسار الكلمة في تلميح الكلمة
- **Genetic Algorithm** — محسّن لتوزيع الحروف
- **Hill Climbing (Local Search)** — لتحسين توزيع الحروف العشوائية
- **Best-First Search** — للبحث السريع في الجريد
- **Minimax + Alpha-Beta** — (متوفر في algorithms.js للتطوير المستقبلي)
